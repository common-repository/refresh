Customize your WordPress login to make it look professional.

== Description ==

REFRESH allows you to customize your WordPress login to make it look professional. You can additionally add your own logo on the top left of the login page.

== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (refresh.zip) on your computer
1. Click "Install Now" and then hit the activate button

== Frequently Asked Questions ==

= Can I add my own logo on the login page ? =

Yes. Go to the login settings page then update it.
